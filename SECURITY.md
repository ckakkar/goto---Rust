# Security Policy

## Supported versions

Only the latest published crate version is guaranteed to receive security fixes.

## Reporting a vulnerability

Please do not open public issues for security-sensitive reports.

Report privately via email:

- cyruskakkar@gmail.com

Include:

- A clear description of the issue
- Reproduction steps or proof of concept
- Impact assessment (what can go wrong)
- Suggested mitigation (if known)

You will receive an acknowledgment as soon as possible, and validated issues will
be fixed and disclosed responsibly.
